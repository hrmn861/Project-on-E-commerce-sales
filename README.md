# 🛒 E-Comersales
### An Interactive E-Commerce Sales Analytics Dashboard

E-Comersales is a Streamlit-based web application designed to help users analyze e-commerce sales data through interactive visualizations and insightful analytics. The dashboard enables users to upload datasets, explore sales trends, visualize key performance metrics, and derive actionable business insights.

---

## 📌 Features

- 📂 Upload CSV and Excel datasets
- 📊 Interactive sales visualizations
- 📈 Trend analysis and business insights
- 🎨 Modern dark-themed dashboard
- 📱 Responsive and user-friendly interface
- ⚡ Fast performance using Streamlit

## 📥 Installation

Clone the repository

```bash
git clone https://github.com/your-username/E-Comersales.git
```

Move into the project directory

```bash
cd E-Comersales
```

Install the required libraries

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```

## 📊 Dataset

The project uses an e-commerce sales dataset containing information such as:

- Profit
